# 栈
- 栈是一种“操作受限”的线性表，只允许在一端插入或删除

## 顺序栈
- 用数组实现的栈
- 入栈，出栈过程中空间复杂度O(1)
- 入栈，出栈过程中时间复杂度O(1)

### 支持动态扩容的顺序栈
- 基本思想
    当数组空间不够时，重新申请一块更大的内存，将原来数组中数据拷贝过去。
    
- 入栈，出栈过程中空间复杂度O(1)
- 入栈，出栈过程中时间复杂度O(1)

## 链式栈
- 用链表实现的栈
- 入栈，出栈过程中空间复杂度O(1)
- 入栈，出栈过程中时间复杂度O(1)

## 特性
- 后进先出

## 常见算法
- 浏览器前进后退
- 四则运算+括号匹配
- 链式栈
